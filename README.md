# Enduro Kyiv

## Про проект

Проект акцентує увагу на продуктивності, плавних анімаціях та зручному користувацькому досвіді на всіх типах пристроїв.

### Ключові функції

*   **Каталог техніки**: Сітка карток послуг з основною інформацією.
*   **Інтерактивний слайдер**: Детальний перегляд моделей із можливістю навігації.
*   **Технічні специфікації**: Відображення характеристик (об'єм двигуна, потужність, тощо).
*   **Зворотний зв'язок**: Валідована форма для швидкого зв'язку з менеджерами.
*   **Повна адаптивність**: Відображення на мобільних телефонах, планшетах та десктопах.
*   **Micro-interactions**: Анімовані ховер-ефекти та плавні переходи для покращення UX.

## Технологічний стек

*   **Core**: React
*   **Styling**: CSS Modules 
*   **Build Tool**: Vite 
*   **Icons**: Lucide React 

## Структура проекту

```text
src/
├── assets/          # Ресурси 
├── components/      # UI-компоненти
├── App              # Кореневий компонент 
└── main             # Точка входу в додаток
```

## Встановлення та запуск

Для локального запуску проекту переконайтеся, що у вас встановлено **Node** (версія 20х).

### 1. Клонуйте репозиторій

```bash
git clone https://github.com/your-username/enduro-kyiv.git
```

### 2. Перейдіть у директорію проекту

```bash
cd enduro-kyiv
```

### 3. Запустіть режим розробки

```bash
npm run dev
```
